**ESLint** 는 ECMAScript/JavaScript 코드에서 버그나 일정함을 체크 하기 위해 사용하는 툴이며, 자바스크립트 문법에서 에러를 제대로 표시해주는 도구이다.

**Prettier**는 코드(JavaScript 뿐만 아니라)를 이쁘게 보이게 하기 위해서 포맷에 맞춰서 정리해주는 도구이다.

나는 인영님의 한줄을 참 좋아한다.

> eslint는 코드 퀄리티를 보장하도록 도와주고, prettier는 코드 스타일을 깔끔하게 혹은 통일되도록 도와준다.

## 🏗 설치

```bash
# ESLint
npm install eslint --save-dev
```

```bash
npm init @eslint/config
```

## ⚙️ 설정

우선 간단하게 설명하자면

Prettier를 Local에서 적용을 시키고, ESLint를 이용하여 코드의 퀄리티를 개선할 때, Prettier까지 적용을 시키며 코드를 수정할 수 있다.

### VS Code에서 미리미리...

프론트엔드 개발자들은 웬만하면 Visual Studio를 사용 한다고 생각한다. VS code의 설정에서 Prettier와 ESLint를 미리 확인 할 수 있는 방법이 있다.

1. Extension으로 ESLint와 Prettier를 다운 받는다.
2. 파일을 수정하고 저장 할 때 마다, ESLint를 확인하고 Prettier를 적용하는 설정을 settings.json에 한다.

```bash
{
  "editor.formatOnSave": true,
  "editor.defaultFormatter": "esbenp.prettier-vscode",
  "editor.codeActionsOnSave": {
    // For ESLint
    "source.fixAll.eslint": true
  }
}
```

### Prettier를 ..

- printWidth: 단어의 최대 갯수 (default: 80)
- tabWdith: 인덴트 indent의 너비 (default: 2)
- useTabs: space 대신 tab을 사용한다 (default: false)
- semi: ; 를 끝에 둔다 (default: true)
- singleQuote: “ “ 가 아닌 ‘ ‘ 를 사용한다. (default: false)

등 더 많은 옵션은 [여기](https://prettier.io/docs/en/options.html) 에서 확인하면 된다.

실제 적용사례

```json
// prettierrc.json
{
  "trailingComma": "es5",
  "tabWidth": 2,
  "semi": false,
  "singleQuote": true
}
```

### ESLint 를 ..!

바로 위에 설치칸에서 eslint config를 만들었기 때문에 큰 설정은 없지만 Prettier plugin을 포함, 다른 plugin을 설정 할 수 있다.
예를 들어, airbnb (eslint-config-airbnb)는 npm 으로 install 하고, extends에 추가만 해주면 된다. [airbnb](https://github.com/airbnb/javascript/tree/master/packages/eslint-config-airbnb)

직접 airbnb 플로그인을 사용하지는 않았지만, Prettier도 .. 넣어야하니껜

```json
{
	...
	"extends": [
    "eslint:recommended",
    "plugin:react/recommended",
    "plugin:@typescript-eslint/recommended",
    "prettier",
    "plugin:prettier/recommended"
  ],
	"plugins": ["prettier"],
	"rules": {
		"prettier/prettier": "error"
	}
}
```

이제 package.json 에 eslint 라는 명령어를 넣어주고 실행을 시키면, eslint 라는 명령어를 가지고 prettier는 잘 적용이 되었는지, eslint는 잘 적용이 되었는지 확인을 해서 더욱 더 이쁜(?) 코드를 확인 할 수 있다.
