MUI 웹사이트에 우연히 이런 그라데이션이 들어 있는 Text를 보게 되었다. 강조하는 글자에 넣어주면 좋을 것 같아서 한 번 응용해봤다.

```html
<span>UI library</span>
```

이런 식으로 가정했을때

```css
span {
  background: linear-gradient(
    to right,
    $ {({theme}) => theme.color.font.primary},
    $ {({theme}) => theme.color.font.primary2}
  );

  background-clip: text;
  -webkit-background-clip: text;
  -webkit-text-fill-color: transparent;
}
```

이렇게 코드를 짜주면 gradient 가 들어 있는 Text를 넣을 수 있다.
