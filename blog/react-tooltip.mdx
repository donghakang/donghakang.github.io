React 에서 tooltip을 넣고 싶을때, custom 하기도 편하고 쓰기에 편한거 같아서 자주 사용하는 기능 중 하나이다.

## Installation

```bash
yarn add react-tooltip
```

## Usage

```jsx
<>
	<div data-tip data-for="tooltip">visible</div>
	<ReactTooltip id="tooltip">information</ReactTooltip>
</>
```

타겟의 `data-for`와 `ReactTooltip` 의 `id`을 간단하게 이어주기만 하면 된다.

- place: 위치를 지정한다 (bottom, top, right, left)
- type: tooltip의 색을 지정한다 (dark, success, warning, error, info, light)
- effect: 움직이는 tooltip, 고정된 tooltip 을 설정한다. (float, solid)

## Customization

의외로 간편하게 css에서 정해주면 된다. 그래도 overwrite되는 것들이기 때문에 !important를 항상 붙여주자
