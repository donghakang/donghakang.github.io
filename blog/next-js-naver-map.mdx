1.  API KEY를 받아온다.

![스크린샷 2022-06-03 오후 3.16.46.png](/img/blog/naver-map2.png)

여기서 API KEY는 Client ID 이며, 약 10자로 된 영문숫자가 섞인 키일 것이다. 이것을 Next.js 프로젝트에서 `.env` 파일에 넣자

```
// .env
NEXT_PUBLIC_NAVER_API_KEY=xxxxxxxxxxx
```

1. `_app.tsx`를 수정한다.

```jsx
import type { AppProps } from 'next/app'
import Script from 'next/script'

function MyApp({ Component, pageProps }: AppProps) {
  return (
    <>
			<Script
        type="text/javascript"
        src={`https://openapi.map.naver.com/openapi/v3/maps.js?ncpClientId=${process.env.NEXT_PUBLIC_NAVER_API_KEY}`}
        strategy="beforeInteractive"
      ></Script>
	    <Component {...pageProps} />
    </>
  )
}

export default MyApp
```
