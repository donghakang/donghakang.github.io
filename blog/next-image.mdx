Next version 10 이상에서 부터는 image 컴포넌트를 사용할 수 있다.

### React 혹은 전 Next 버젼

```jsx
...
<div>
	<img src="/img/image.png" alt=""/>
</div>
...
```

### Next 10 이상 버전

```jsx
import Image from 'next/image'

...
<div>
	<Image src="/img/image.png" alt="" width={200} height={200}/>
</div>
...
```

1. Web 스피드를 더 향상 시켜주고 optimization을 자동으로 시켜준다.
2. 자동으로 Responsive 하게 만들어준다.
3. Image 가 scroll로 인해서 보여질 때만 render 해줘서 더 빠르게 Page가 실행된다.
4. CLS(Cumulative Layout Shift: 갑자기 레이아웃이 변경되는 상황)을 방지 해준다.

---

## 그 외의 props

### Layout

layout은 이미지가 responsive 하게 될 수 있도록 도와준다.

- fill: 전체
- fixed: 고정
- intrinsic: 상위 parent와 같게
- responsive

### Loader

url 에서 이미지를 가져와야할 때, src에 직접적으로 쓰지 않고 loader라는 url 을 담는 function을 넣으면 url에 맞는 이미지를 가져온다.

### Placeholder

load가 되기전, 이미지가 어떻게 보여질지 보여준다.

- blur: 이미지가 흐릿하게 보인다.
- empty: default

### styling

css 에서 주의할 점은 Image 같은 경우 `style={{}}` 안에 스타일을 넣을 수 없고, 그냥 className으로 스타일을 가져오는 방법을 사용하자.

### priority

lazy load를 도와준다 (스크롤을 할때, 이미지를 불러오는 기능). 이것은 default로 되어있다.

- lazy: default, lazy load
- eager: 그냥 초기에 불러온다 (optimization에 그렇게 좋지는 않다).
