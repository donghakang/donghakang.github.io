개인블로그 는 개발자라면 자기의 웹사이트를 가지고 있어야 하지 않나 라는 생각에서 시작된 개인 프로젝트 입니다. 

### 👨🏻‍💻 **개인의 역할**

- Client-side rendering 보다 더 빠른 초기 진입을 위해 `Next.js` 를 이용하여 Server-side rendering 경험
- `Firebase/Google Analtics` 를 이용해 접속자 수 확인
- 많은 사용자들이 핸드폰으로 접근 한다는 점에서, `Responsive Web Design` 으로 웹 디자인 구현
- `ESLint`  `Prettier` `Commitlint` 를 이용하여 문법확인
- `Three.js`, `emotion` , `framer-motion`를 활용하여 시각적인 효과 및 반응형 웹 구현
