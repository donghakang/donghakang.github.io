## 🎉 Pulmorad 란?

![DICOM Web Viewer](/img/pulmorad.png)
**Pulmorad**는 React 기반 웹사이트로서 의료이미지(DICOM)를 오픈하고 이미지를 변형 할 수 있게해주는 툴입니다. OHIF Viewer라는 기존에 있는 오픈소스를 사용했으며, 의사들이 필요한 추가적인 기능을 추가하고, 서버와 데이터 베이스를 연결하는 일을 주로 맡아서 했습니다. 

### 👨🏻‍💻 **개인의 역할**

- `Material-UI`를 이용하여 이미지 변화에 관련된 UI 개발로 프론트엔드를 경험
- `Fastapi`, `Postgres`를 사용하여 백엔드 기능도 구현
- 전반적인 웹에 대한 경험 뿐만 아니라, 다양한 라이브러리와 프레임워크를 사용하여 현재 트렌드와 다양한 문제를 해결
[Demo](https://github.com/donghakang/Pulmorad)
