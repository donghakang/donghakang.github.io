**DICOM Web Viewer**는 Pulmorad를 이용해 제작한 개인 프로젝트이며, 의료이미지(DICOM 파일)을 더 빠르고 쉽게 볼 수 있게 해주는 데스크탑 어플리케이션으로도 사용할 수 있는 뷰어 입니다. 현재 대다수의 오픈소스 DICOM 뷰어와는 다르게 다양한 기능(이미지 변형, 프리셋 기능, Pixel 데이터가 없는 이미지 열람 등)이 탑재 되어있으며
이로 인해 뉴욕에 있는 [MD.ai](http://MD.ai) 라는 메디컬 이미징 회사에서 같이 프로젝트를 하자는 제안도 받았습니다. 

### 👨🏻‍💻 **개인의 역할**

- 디자인/기획 부터 직접 구현
- 데스크탑 어플리케이션으로 구현하기 위해 PWA 사용
- redux-toolkit 을 사용한 상태관리
- 다양한 라이브러리와 프레임워크 (`emotion`, `React`, `Material-UI`, `framer-motion` 등)을 사용하여 UI/UX를 구현
